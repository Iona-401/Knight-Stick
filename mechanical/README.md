
# Mechanical Design

This directory contains the mechanical design files for the **Knight Stick**
flight stick. All mechanical components are designed to be **3D printable** and
are intended to integrate with the custom electronics.

The mechanical design was created in **Onshape** and exported for distribution
and fabrication.

## Contents

- 3D model exports (e.g., STEP and STL files) for individual parts and assemblies
- Mechanical drawings and documentation (where applicable)
- Visual assets such as renders or reference images

## 3D Printing Notes

- Parts are intended for consumer-grade FDM 3D printers.
- Recommended print settings (material, layer height, infill, orientation)
  should be documented here as the project matures.
- Verify tolerances before final assembly, especially for moving parts.
